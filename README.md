## ğŸ”§ Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/InshaHabib/InshaHabib-Portfolio.git
   cd InshaHabib-Portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Start development server**
   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. **Build for production**
   ```bash
   npm run build
   # or
   yarn build
   ```
